# Estudos Selenium
## Sobre o Projeto
Este repositório foi criado com o objetivo de aprofundar e aplicar conhecimento em automação de testes, utilizando o Selenium
como principal ferramenta. O projeto é parte do meu aprendizado contínuo em qualidade de software e integra práticas estudadas
no curso de Testes Automatizados.
---
## Objetivo
- Explorar recursos do Selenium WebDriver;
- Implementar padrões como Page Object, Page Factory e Builder;
- Integrar com ferramentas como JUnit e Maven;
- Estruturar uma base sólida com BaseObejct e BaseTest.
---
## Tecnologias Utilizadas
- **Selenium WebDriver:** Automação de interação web;
- **Java:** Lógica dos teste;
- **JUnit:** Estrutura de asserções;
- **Maven:** Gerenciamento de dependência e build;
- **Git:** Controle de versão.
